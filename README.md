# AI Knowledge Base

A designer-friendly knowledge base for the AI Experiences Design Team.

## Features

- 📚 Searchable concepts library
- 👥 Team directory with expertise areas
- ✏️ Easy "Add Entry" form (no coding needed!)
- 📱 Fully responsive
- 🎨 Clean, modern design

## Quick Start

1. Open `index.html` in your browser
2. Click "+ Add Entry" to create new concepts
3. Edit `knowledge-base-data.json` to add content

## Deployment

### Vercel (Recommended)

1. Go to [vercel.com/new](https://vercel.com/new)
2. Import this repository
3. Deploy!

Your URL will be: `your-project.vercel.app`

### Custom Domain

Add a subdomain like `kb.aitools-hub.de` in Vercel settings.

## Contributing

Use the "+ Add Entry" button in the UI to generate JSON for new entries.

## Tech Stack

- Pure HTML/CSS/JavaScript
- No build process needed
- Works offline
